# 🤖 Chat With PDF - Streamlit App

### A Streamlit-based chatbot that allows users to upload PDFs and chat with them using AI-powered retrieval and response generation.

### 🚀 Features

- 📂 Upload Multiple PDFs

- 🔍 AI-Powered Search & Response

- 🧠 Uses Llama3-8B via Groq API

- 📚 Embeds Documents for Better Context
